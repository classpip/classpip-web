DASHBOARD

La aplicación Dashboard es la herramienta de gestión del profesor, dónde puede configurar todos los juegos, inscribir alumnos y generar los grupos pertinentes con ellos, añadir o descargar recursos, entre otras funciones. El Dashboard es la base de todo.


APLICACIÓN MÓVIL PARA ALUMNOS

Sin la aplicación móvil para el alumno, este no tendría forma de jugar a los juegos propuestos por el profesor. De uso sencillo e intuitivo, esta aplicación no solo permite a los usuarios jugar a los juegos que propone el profesors sino que además permite personalizar el perfil del alumno asi como tener un control sobre los grupos en los que esta inscrito.

CLASSPIP EXPRESS

Classpip Express es la aplicación de juego rápido que permite al profesor generar, des del Dashboard, juegos que no requieren al alumno estar registrado en la plataforma para poder jugar, únicamente con introducir el nombre de usuario y la clave del juego que ha generado el profesor, el alumno podrá participar en la actividad.
